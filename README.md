# Калькулятор кредиту та депозиту
### Лабораторна робота №1 — Варіант 3

Застосунок розроблено на **Flutter (Dart)** з використанням **Material Design 3**.


## Опис

Мобільний застосунок для розрахунку параметрів кредиту та депозиту з можливістю збереження результатів у текстовий файл.


##  Функціональність

### Кредитний калькулятор
| Режим                | Опис                                                   |
|----------------------|--------------------------------------------------------|
| **Ануїтетний**       | Фіксований щомісячний платіж протягом усього терміну   |
| **Диференційований** | Зменшуваний платіж (основний борг ÷ термін + відсотки) |

**Вхідні дані:**
- Сума кредиту (UAH)
- Річна відсоткова ставка (%)
- Термін (місяці)
- Тип кредиту

**Результати:**
- Щомісячний платіж
- Загальна сума виплат
- Загальна сума відсотків
- Переплата у % від суми кредиту
- Детальний графік платежів (по місяцях)
- Кругова діаграма структури виплат


### Депозитний калькулятор
| Режим          | Опис                                                        |
|----------------|-------------------------------------------------------------|
| **Складний %** | Капіталізація щомісяця (відсотки нараховуються на відсотки) |
| **Простий %**  | Відсотки нараховуються лише на початкову суму               |

**Вхідні дані:**
- Початкова сума (UAH)
- Щомісячне поповнення (UAH, може бути 0)
- Річна відсоткова ставка (%)
- Термін (місяці)
- Тип нарахування

**Результати:**
- Кінцева сума
- Загальна сума нарахованих відсотків
- Загальна сума внесків
- Ефективна річна ставка (CAGR)
- Графік нарахування (по місяцях)
- Кругова діаграма структури доходу


## Математичні формули

### Ануїтетний кредит:
```
M = P × r × (1+r)ⁿ / ((1+r)ⁿ - 1)
```
де: `P` — сума кредиту, `r` — місячна ставка = річна/12/100, `n` — кількість місяців

### Диференційований кредит:
```
Dᵢ = P / n                        (основний борг щомісяця)
Iᵢ = (P - Dᵢ × (i−1)) × r        (відсотки місяця i)
Mᵢ = Dᵢ + Iᵢ                      (платіж місяця i)
```

### Складний відсоток (FV з поповненням):
```
балансᵢ = балансᵢ₋₁ × (1 + r) + поповнення
```

### Проста ставка:
```
відсотки_місяця = початкова_сума × r
кінцева = початкова + внески + відсотки × n
```

### Ефективна річна ставка (CAGR):
```
r_eff = (FV / PV) ^ (1/years) - 1
```


## Збереження файлів

Результати зберігаються у папку документів застосунку:
- **Android/iOS:** `getApplicationDocumentsDirectory()`
- **Desktop:** `getDownloadsDirectory()`

Формат імені файлу: `credit_YYYYMMDD_HHMMSS.txt` / `deposit_YYYYMMDD_HHMMSS.txt`

Файл містить:
1. Вхідні параметри
2. Підсумкові результати
3. Повний графік платежів / нарахувань у форматі таблиці


## Запуск застосунку

### Вимоги:
- Flutter SDK >= 3.10.0
- Dart SDK >= 3.0.0

### Встановлення:
```bash
flutter pub get
flutter run
```

### Збірка APK:
```bash
flutter build apk --release
```

### Збірка для Desktop (Windows):
```bash
flutter build windows
```
